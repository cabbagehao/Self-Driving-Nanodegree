# Self-driving-Car  
包含Udacity无人车学位3个学期的项目和练习.  

### Term1:  
  1. 车道线检测  
  2. 交通灯检测  
  3. 行为克隆  
  4. 高级车道线检测  
  5. 车辆探测  
### Term2:  
  1. 传感器融合: EKF扩展卡尔曼滤波  
  2. 传感器融合: UKF无迹卡尔曼滤波  
  3. 车辆定位: 粒子滤波  
  4. 控制: PID  
  5. 控制: MPC  
### Term3:  
  1. 路径规划: 高速路变道行驶  
  2. 车道识别: 语意分割  
  3. 毕业项目: ROS开发,通过真车场地测试及红绿灯检测.
